GitHub and Git

22nd August 2020
1 minute read
Tweet
Curated list of online resources referred during the past when doing OSS work using GitHub and git.

Software & Tools

  • Git and GitBash
  • GitHub CLI to try GitHub on the terminal
  • GitHub Desktop app
  • Paper app for markdown editing
VS Code Extensions
  • Git Lens by Eric Amodio annotates valuable git insights seemlessly on the code editor
  • Git Graph by Michael Hutchison visualize git activities of the repository
  • Git History by Don Jayamanne shows the log, file history, and compare branches or commits
  • Git Blame by Wade Anderson shows the last edited author in the statusbar
  • GitHub for VS Code allows you to review and manage pull requests and issues

Conventional Commits

  • The official conventional commits guide
  • The original guide introduced by the Angular team
  • Git commit message conventions with examples discusses the objectives and examples of commit messages
  • Link issues and pull requests using commit messages is useful to close issues when the pull requests are merged
CLI Tools
  • Standard Version is a CLI tool by Conventional Changelog used to bump-up version according to semver and to generate CHANGELOG based on conventional commits
  • Commit Lint is another CLI tool to verify commit messages and suggest corrections

Documentation

  • Emoji cheat sheet generated via GitHub Emoji API
  • Bookworm repository has some templates for CONTRIBUTING.md, README.md, and .gitignore
  • VS Code, Flow, Babel, Parcel JS Bundler repos also have good documentation
  • Contributor Covenant for Code of Conduct
  • Open Collective for sponsorships
  • Flow and Babel sites for Usage and Getting Started Guides

Videos

  • More Git & GitHub Secrets by Zach Holman reveals hidden and not so hidden tips and tricks

Courses

  • Free Classes by GitHub Training & Guides is playlist with series of how-tos on various GitHub features

Community

  • Git and GitHub Secrets has awesome list of tips and tricks
  • Git and Gitflow Cheat Sheet is an awesome list of git commands
  • Git Hooks Awesome List contains everything need to know about hooking up custom scripts

1 minute read

Curated list of online resources referred during the past building and deploying github sites using jekyll, liquid, and markdown.

Dev Tools

  • Hello world article has the links to all the dev tools

Plugins

  • Docsify JS is a JS-based alternative approach to Jekyll
    • Contributors hangout at discord.com
  • Shields.io is the go-to place for badges
  • Jekyll Emoji to render emojis on GitHub sites
    • _emoji-optimize.scss contains inline content related fixes
  • Speaker Decks to embed powerpoint slides
    • _speaker-deck.scss maintains the aspect ratio
  • Twitter Button for social sharing
    • _social-icons.scss for making it looks better

Coding

  • Cheat sheet for Jekyll has a lot of Liquid syntax
  • Jekyll code snippets by Mark Otto is a good source of how to do this and that
  • SCSS with Jekyll by Mark Otto is the best guide for themes
  • CSS dark mode article on how to use prefers-color-scheme
  • Host Web Apps and Images article on how to use GitHub as a CDN
General Guides
  • CodeGuide for HTML and CSS
  • WTF? HTML and CSS has some tips and tricks
  • SCSS style guide article on CSS tricks is also a good source

Themes

  • WTF? HTML and CSS
  • Poole
  • Hyde is a two-column theme based on Poole

Courses

  • GitHub Actions by Aaron Stewart
  • Customize GitHub Actions by Enrico Campidoglio
  • GitHub Workflow by Aaron Stewart
  • Coding with GitHub Learner path on Pluralsight
